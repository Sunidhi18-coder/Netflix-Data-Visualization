# 📊 Netflix Data Visualization

## 📌 Project Overview
This project analyzes the Netflix dataset using Python and creates visualizations to understand trends in Netflix content.

## ✨ Features
- Movies vs TV Shows Comparison
- Content Ratings Analysis
- Movie Duration Analysis
- Release Year Analysis
- Top 10 Countries by Number of Shows

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
- netflix_titles.csv

## 📈 Output
The project generates the following visualizations:
- Movies_vs_tvshows.png
- Movies_Duration.png
- Release_Year.png
- Top_10_Countries.png
- content_ratings.png

## 🚀 How to Run

```bash
pip install pandas matplotlib
python Netflix_Data_visualization.py
```
